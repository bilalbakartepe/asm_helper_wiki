Get Started

Assembly Helper projesi: Temel 8086 Mimarisi Assembly talimatlarının kullanımını öğrenmek ve anlamak isteyenler için tasarlanmış bir projedir. Proje başlangıç aşamasında olup geliştirilmeye devam edilecektir.

Programın kullanılabilmesi için kayıt oluşturmanız gerekmektedir. Kullanıcı kayıtı oluşturduktan ve giriş yaptıktan sonra proje çalıştırabilecek hale gelebilecektir.

Login Page

Program ilk açıldığında bir giriş sayfası ile karşılaşıyorsunuz. İlk deneyiminiz olduğunu varsayarak bu aşamada sağ alt bölümde bulunan I do not have an account yazısına tıklıyoruz.

Get A Global Account

Gerekli bilgileri doldurup satın al butonuna tıklıyoruz. Bu işlem için İnternet gerekmektedir. İşlem gerçekleşti ise Let's create an account butonuna tıklayınız.

Create User Page

Gerekli bilgileri doldurunuz ve kaydet butonuna tıklayınız. Ardından Giriş Sayfasına gidiniz

Kullanıcı bilgileriniz ile giriiş yaptıktan sonra Editor Sayfasına ulaşmış olacaksınız

Editor Page

Text alanına assembly talimatları yazabilirsiniz ve RUN butonu ile debug penceresine geçebilir veya File menüsünden proje ve dosya işlemleri yapabilirsiniz.

Debug Page

Bu pencerede register değerlerini Hexadecimal olarak set edebilir, talimatları adım adım veya direkt çalıştırabilirsiniz.
